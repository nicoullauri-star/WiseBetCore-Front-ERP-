# WiseBet Lab | Analytics Terminal V13

Terminal de analítica avanzada para el seguimiento de ROI y gestión de banca en apuestas deportivas. Diseñado para traders profesionales y gestores de planes de inversión.

## Características Principales
- **Sistema Multiprograma**: Gestión independiente o consolidada de planes (ELITE, PREMIUM, STANDARD, PERSONAL).
- **Analítica en Tiempo Real**: Cálculo instantáneo de ROI, Yield, Drawdown Máximo, Ratio de Sharpe y Criterio de Kelly.
- **Visualización Avanzada**: Gráficos interactivos de evolución patrimonial, hit rate por cuotas y comparativa de eficiencia.
- **Importación Inteligente**: Motor de parsing robusto para picks simples y combinados (doubles).
- **Diseño Premium**: Interfaz glassmorphism optimizada para alta densidad de datos.

## Instalación y Uso Local

### Requisitos
- Node.js (v18+)
- Navegador moderno

### Pasos
1. **Instalar dependencias**:
   ```bash
   npm install
   ```
2. **Configuración**:
   - El código de acceso por defecto es `admin123`.
   - Puedes configurar variables de entorno en `.env.local` si integras servicios adicionales.
3. **Ejecutar**:
   ```bash
   npm run dev
   ```
   Accede a `http://localhost:3000`.
